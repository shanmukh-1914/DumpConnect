# DumpConnect — Smart Waste Management Prototype

This repository is a frontend prototype built with React + Vite demonstrating a Smart Waste Management System for municipal corporations and citizens.

Features in this prototype:
- User and Admin dashboards
- Live truck tracking simulation on a Leaflet map
- Pickup request flow using browser Geolocation
- Area threshold simulation and dispatch logic (frontend)
- Charts and analytics (Recharts)
- Tailwind CSS, Zustand state management, Framer Motion animations

Run locally:

```bash
npm install
npm run dev
```

Open the site at http://localhost:5173

Notes:
- This is a frontend-only prototype. No backend APIs are required.
- Data is simulated using files in `src/mockData` and runtime intervals.

Explore:
- Public pages: Landing, About, Features
- Auth pages: Login / Signup (role selection)
- User: Dashboard, Live Map, Request Pickup
- Admin: Dashboard, Live Monitoring, Reports

Enjoy exploring the DumpConnect prototype!
